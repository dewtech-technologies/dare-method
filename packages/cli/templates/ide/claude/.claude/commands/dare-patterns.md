# Comando: /dare-patterns

> **Equivalente no terminal:** `dare patterns --ai`


Descobre padrões recorrentes no codebase (`dare patterns`) — determinístico, sem LLM.

```bash
dare patterns
dare patterns --check
dare patterns --ast
dare patterns --dir ./packages/api
dare patterns --inject
```

Gera `DARE/patterns-facts.json` + `DARE/PATTERNS.md`. Complete com `/dare-dna`.
